## ✨ Usage Guidelines

### When to Use Each Color

| Color | Usage | Don't Use For |
|-------|-------|---------------|
| **Soft Leaf Green** | Primary buttons, active states, trust | Long text blocks |
| **Fresh Mint** | Secondary emphasis, success, highlights | Backgrounds |
| **Light Yellow** | Subtle accents, backgrounds, highlights | Important text |
| **Sky Blue** | Information, secondary actions | Primary CTAs |
| **Soft Lavender** | Premium badges, special indicators | Body text |
| **Blossom Pink** | Completion, achievement, celebration | Primary interaction |
| **Cream White** | Backgrounds, cards, surfaces | Text on light bg (use Forest instead) |
| **Soft Sand** | Secondary backgrounds | Foreground elements |
| **Deep Forest** | Primary text, headings, depth | Backgrounds (too dark) |
| **Butter Yellow** | Alerts, warnings, important | Regular text |
| **Petal Pink** | Celebration, rewards, special | Standard interaction |

## 📐 Opacity Levels

### Standard Opacity Values
- **100% (opaque)**: Solid colors, primary text, CTA buttons
- **90% (0.9)**: Card backgrounds, strong elements
- **70% (0.7)**: Secondary text, hover states
- **60% (0.6)**: Muted text, disabled states
- **50% (0.5)**: Borders, subtle dividers
- **40% (0.4)**: Very subtle text, placeholder text
- **30% (0.3)**: Shadows, soft glows
- **20% (0.2)**: Very light shadows, barely visible borders
- **15% (0.15)**: Faint backgrounds, overlay hints
- **12% (0.12)**: Extremely subtle contexts, sidebar tints
- **10% (0.1)**: Barely perceptible accents
- **8% (0.08)**: Minimal visual hints
- **5% (0.05)**: Almost invisible, texture only

**Example**: `rgba(76, 175, 106, 0.15)` = 15% opacity green

## ♿ Accessibility Notes

### Contrast Ratios
- ✅ **WCAG AAA** (7+ ratio): #4CAF6A text on #FAFFF7
- ✅ **WCAG AAA** (7+ ratio): #1E2A22 text on #F5F9C1
- ✅ **WCAG AA** (4.5+ ratio): All colors meet AA minimum
- ✅ **WCAG AAA** (7+ ratio): Most primary combinations

### Color Blind Safe
- ✅ Green (#4CAF6A) visible to deuteranopia
- ✅ Blue (#8ED6FF) universally visible
- ✅ Use patterns/icons + color for indication
- ✅ Never rely solely on color to convey meaning

### Recommended Text Combinations
| Text Color | Background Color | Ratio |

## 🔄 Color References in Code

### CSS Variables
```css
/* Light Mode */
--spring-leaf: #4CAF6A;
--spring-mint: #7ED957;
--spring-forest: #1E2A22;

/* Dark Mode */
--spring-leaf-dark: #6DD477;
--spring-mint-dark: #90FF6B;
--spring-forest-dark: #E8F5E9;
```

### Tailwind Classes
```tsx
// Light mode (default)
text-spring-leaf          // #4CAF6A
bg-spring-mint            // #7ED957
border-spring-forest      // #1E2A22

// Dark mode
dark:text-spring-leaf-dark    // #6DD477
dark:bg-spring-mint-dark      // #90FF6B
dark:text-spring-forest-dark  // #E8F5E9

// With opacity
bg-spring-leaf/12         // 12% opacity green
text-spring-forest/70     // 70% opacity text
```
